Hoe werkt de verificatie?

Deze website staat op GitHub Pages. Elke nieuwe voorspelling wordt toegevoegd via een Git-commit. De commit-timestamp is publiek zichtbaar en kan niet achteraf worden aangepast zonder dat dit zichtbaar is in de history.

Bovendien bevat elke voorspelling een SHA-256 hash van de tekst JAAR|TYPE|NAAM|DATUM. Die hash kun je zelf narekenen om te controleren dat de gegevens niet zijn gewijzigd.
